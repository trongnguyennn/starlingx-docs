# Chương 3: Cài đặt Nền tảng

Chương này mô tả quy trình cài đặt StarlingX từ boot đến cấu hình bootstrap.

## Nội dung chính

- Khởi động Controller-0 từ USB/ISO.
- Cài đặt hệ điều hành cơ sở StarlingX OS.
- Tạo và xác thực file cấu hình Bootstrap (`localhost.yml`).
- Thực hiện cấu hình ban đầu và kiểm tra trạng thái cluster.

## Mục tiêu chương

1. Hoàn tất cài đặt controller đầu tiên.
2. Thiết lập cấu hình bootstrap đúng chuẩn.
3. Xác minh cluster hoạt động và sẵn sàng thêm node.
