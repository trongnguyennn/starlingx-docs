# What is StarlingX

StarlingX is an open-source distributed cloud platform optimized for edge computing and telecommunications workloads. It integrates virtualization, container orchestration, storage, and networking to deliver a resilient platform for production edge deployments.

## Key characteristics

- Designed for low-latency, high-availability edge environments.
- Combines KVM, Kubernetes, and Ceph for compute, orchestration, and storage.
- Focus on deterministic operations, lifecycle management, and observability.
